## Vigolium

[Vigolium](https://docs.vigolium.com/) 是一款用 Go 编写的高精度 Web 漏洞扫描器。它将确定性的模块化扫描与 AI 驱动的 Agent 分析相结合，提供广泛而深入的 Web 应用安全覆盖。
